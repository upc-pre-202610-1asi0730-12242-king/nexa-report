# Capítulo III: Requirements Specification

## 3.1. User Stories

Las User Stories de Nexa consolidan las necesidades identificadas en Lean UX, entrevistas, needfinding y diseño del producto. La organización por épicas permite relacionar el recorrido público del Landing Page, el portal B2B y las operaciones internas con los segmentos objetivo definidos para el proyecto.
